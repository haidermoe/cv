# موقع السيرة الذاتية لحيدر محمد

موقع شخصي بسيط وحديث يعرض سيرتك الذاتية بالعربية، مبني باستخدام HTML و CSS فقط (بدون إطار عمل).

## طريقة التشغيل

1. افتح المجلد `my cv` على سطح المكتب.
2. اضغط مرتين على ملف `index.html` لفتحه في المتصفح (يفضل Chrome أو Edge).

## تجهيز الملفات

- **الصورة الشخصية**  
  انسخ ملف صورتك:

  `photo_2026-03-05_19-52-12.jpg`

  وضعه داخل نفس المجلد الذي يحتوي على `index.html`.  
  إذا كان اسم الصورة مختلفًا، إمّا:
  - غيّر اسم الملف ليتطابق مع الاسم في الكود، أو
  - عدّل وسم الصورة في `index.html` (`src="..."`) لاسم ملفك الفعلي.

- **ملف السيرة الذاتية PDF**  
  من الأفضل نسخ ملف الـ PDF إلى هذا المجلد بنفس الاسم التالي:

  `HAIDER-M-SHAWKAT-CV.pdf`

  أو عدّل رابط زر "تحميل السيرة الذاتية PDF" في `index.html` إلى اسم الملف الذي تريده.

## التخصيص

- يمكنك تعديل النصوص والألوان من خلال الملفين:
  - `index.html`
  - `styles.css`
- الموقع مضبوط باتجاه من اليمين إلى اليسار (`dir="rtl"`) وخط عربي جميل (`Cairo`).
